    • 多元赋值

      x, y = y, x
      等式俩边是元祖

      多重赋值PI, r = 3.14159, 3的本质由两个步骤构成：
      temp = 3.14159, 3 # 元组打包（Tuple packing）
      PI, r = temp # 序列解包（Sequence unpacking），多重赋值有时也称为序列解包
